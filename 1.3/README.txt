1. Dockerfile и файл с СУБД кладем в одну папку. После этого заходим в командную струку в открытой директории и вводим следующие команды:
docker build -t test_image:latest . (создает образ)
docker run -d -p 9019:5432 --name test_container test_image:latest (поднимает контейнер и прописывает порты для подключения к СУБД)


2. Команда для взаимодействия с СУБД - docker exec -it test psql -U postgres

3. Так же можно запустить СУБД через команду docker-compose up в командной строке корневой директории
