Сущностями в БД "Библиотека" являются читатели, книги, издательства и выданные книги
Для данной БД достаточно связи между сущностями "один к одному" во всех элементах кроме номера читательского билета, который необходим для индентификации читателяю. Остальные атрибуты нужны в сущности "Выдано" для боле полной характеризации выданной книги
